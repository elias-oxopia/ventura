# TextInput

A TextInput can be used to let the user insert text using key strokes. A crucial form item for building interactive apps.

> Form elements do not have `onChange` or `value` props in favour to form control libraries (like [react-hook-form](https://react-hook-form.com)) that use the ref prop to detect the current value.

## Best practices

- Inputs should be wrapped in a FormField component or should be preceded by another component component serving as a label // TODO Link to FormField
