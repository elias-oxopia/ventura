# Icons

The full set of icons from [Feather Icons](https://feathericons.com).

All icons have their own component. Each component name is a pascal cased version of the [Feather Icon](https://feathericons.com).

## Basic usage

  <ArrowDownCircle />

  <Minimize />
  <Minimize2 />
  <Figma />
  <LifeBuoy />
  <XCircle />
  <X />
</Playground>

## Changing styles

There are a few ways this can be done. The default styles can be changed using a theme.

To apply custom styles using CSS in JS, you can use the styles property.

<Playground>
  <GitHub />
  <GitHub style={{width: '24px', height: '24px', stroke: '#116ab8'}}/>
</Playground>

You apply custom styles using CSS classes, you can use the className property.

```CSS
  .blueIcon {
    width: 24px;
    height: 24px;
    stroke: #116ab8;
  }
```

<Playground>
  <GitHub />
  <GitHub className="blueIcon"/>
</Playground>

## Changing icon color

Feather Icons opens up a `color` property to change the color of an icon. We reccomand styling icons using class name or CSS in JS.

<Playground>
  <GitHub color={'#000000'} />
  <GitHub color={'#fdda24'} />
  <GitHub color={'#ef3340'} />
</Playground>
