# Table

A table displays data in a familiar structure, making it easy to read.

## Best practices

Use the different building block to correctly structure your table:

- `Table`
- `Table.Header`
- `Table.HeaderCell`
- `Table.Body`
- `Table.Row`
- `Table.Cell`
