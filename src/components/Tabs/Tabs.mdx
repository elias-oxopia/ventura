# Tabs

Tabs organize and allow navigation between groups of content that are related and at the same level of hierarchy.

## Examples

### Basic usage

<Playground>
  {() => {
    const [currentTabIndex, setCurrentTabIndex] = React.useState(0)
    return (
      <Tabs currentTabIndex={currentTabIndex} onChange={(tabIndex) => setCurrentTabIndex(tabIndex)}>
        <Tab title="First" index={0}>This is the first tab</Tab>
        <Tab title="Second" index={1}>This is the second tab</Tab>
        <Tab title="Third" index={2}>This is the third tab</Tab>
      </Tabs>
    )
  }}
</Playground>
