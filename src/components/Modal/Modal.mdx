# Modal

A modal is a dialog that appears on top of the main content to require user interaction. This modal disables the main content until the user explicitly interacts with it.

## Best practices

- Wrapping your buttons inside `Modal.Footer` will allow them to be automatically displayed right

## Examples

### Basic usage

<Playground>
  {() => {
    const [isOpen, setIsOpen] = React.useState(false);
    return (
      <>
        <Button onClick={() => setIsOpen(true)}>Open basic modal</Button>
        <Modal.Dialog title="This is the title" isOpen={isOpen}>
          <span>Insert your text or form elements here</span>
          <Modal.Footer>
            <Button type='secondary' onClick={() => setIsOpen(false)}>Cancel</Button>
            <Button onClick={() => 'do some logic here'}>Create</Button>
          </Modal.Footer>
        </Modal.Dialog>
      </>
    );
}}

</Playground>
